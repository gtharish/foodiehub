## 📖 About This Project

Have you ever opened your fridge and stared at it, not knowing what to cook? Or felt "sad" and just wanted some comfort food? 

**FoodieHub** solves that problem. It is a modern website built with **React** (the most popular web tool today) that helps you find the perfect meal in seconds. It’s fast, looks good on your phone, and has a Dark Mode for late-night snacking.



## ✨ What Can It Do?

### 1. 🎭 Mood Selector
Tell the app how you feel, and it gives you a suggestion.
* **Happy?** Get a light, fun meal.
* **Sad?** Get warm, comforting food (like chocolate!).
* **Broke?** Get cheap recipes that cost almost nothing (perfect for students).
* **Energetic?** Get healthy, protein-rich food.

### 2. 🥦 Fridge Finder (Smart Logic)
This is a special feature. You select the ingredients you already have at home (like "Egg", "Milk", or "Tomato"), and the app instantly filters the recipes to show you **only** what you can cook right now. No more grocery runs!

### 3. 🌗 Dark Mode
With one click, the whole website switches to a dark color theme. It saves battery and is easier on your eyes at night.

### 4. ⚡ Super Fast
The website loads instantly because it uses **Vite**, a modern tool that makes React extremely fast.

---

## 🛠️ Technologies Used

* **React.js:** To build the website components.
* **Tailwind CSS:** To style the website and make it look professional.
* **Vite:** To run the project quickly.
* **Lucide Icons:** For the clean, modern icons you see.
* **Framer Motion:** For smooth animations when you click buttons.
